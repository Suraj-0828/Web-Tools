6. position: absolute, relative, fixed, sticky me kya
-> relative ->	Normal position ke respect me move hota hai

-> absolute ->	Parent ke relative move hota hai, agar koi relative parent na ho to pura page ka reference leta hai

-> fixed ->  Yeh viewport pe fix ho jata hai, scroll karne pe bhi wahi rehta hai

-> sticky ->  Scroll hone pe ek certain position tak stick karta hai, uske baad normal behave karta hai


7. display: inline, block, inline-block, flex, grid me kya
-> inline ->	Sirf jitna space chahiye utna leta hai, width set nahi kar sakte

-> block ->	Puri width le leta hai, naye line pe chala jata hai

-> inline-block -> 	Inline ki tarah behave karta hai but height/width set ho sakta hai

-> flex ->	Flexible layout ke liye use hota hai

-> grid ->	Grid-based layout design ke liye
